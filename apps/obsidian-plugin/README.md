# Finance Superbrain Obsidian Plugin

This plugin keeps the Finance Superbrain sync ledger visible inside Obsidian.

Build:

```bash
npm --workspace @finance-superbrain/obsidian-plugin run build
```

Install:

1. Build the plugin.
2. Copy `manifest.json`, `main.js`, and `styles.css` into:
   `XAUUSD-Brain/.obsidian/plugins/finance-superbrain/`

The plugin writes a generated note at `Finance Superbrain/Project/Obsidian Plugin Sync.md`.
